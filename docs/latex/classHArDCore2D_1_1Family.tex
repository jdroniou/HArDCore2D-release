\subsubsection*{template$<$typename Basis\+Type$>$\newline
class HAr\+DCore2\+D\+::\+Family$<$ Basis\+Type $>$}

\mbox{\hyperlink{classHArDCore2D_1_1Family}{Family}} of functions expressed as linear combination of the functions of a given basis

If $(f_1,...,f_r)$ is the basis, then the family is $(\phi_1,...,\phi_l)$ where $\phi_i = \sum_j M_ij f_j$. The matrix $M$ is the member m\+\_\+matrix.
