\doxysubsection{Detailed Description}
Base class for global DOF spaces. Provides functions to manipulate global DOFs (the local version being provided by \mbox{\hyperlink{classHArDCore2D_1_1LocalDOFSpace}{Local\+DOFSpace}}). 

The DOFs are organised by increasing geometric entities dimensions\+: DOFs of vertices, DOFs of edges, DOFs of cells.
