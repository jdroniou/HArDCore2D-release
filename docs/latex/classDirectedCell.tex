\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classDirectedCell}{Directed\+Cell}} class is the defining element of a \mbox{\hyperlink{classDirectedGraph}{Directed\+Graph}}. It is defined by a vector of \mbox{\hyperlink{classDirectedEdge}{Directed\+Edge}}\textquotesingle{}s, and a vector of unsigned int\textquotesingle{}s describing the fine cells that make up the cell upon coarsening. The class contains functions related to cell manipulation.
