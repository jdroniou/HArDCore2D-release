Replace sections of vector V by values from vector Z into vector V; the sections are determined by \textquotesingle{}sec\textquotesingle{}. The vectors are any (identical) types of Eigen\+::\+Vector. 

Returns a vector of same length as V, whose segments starting at sec\mbox{[}i\mbox{]}.first and with length sec\mbox{[}i\mbox{]}.second are replaced using the values (successively) of Z. Can be used to\+:

1) create a map from DOFs to unknowns\+: starting from a vector V of -\/1 of size nb of DOFs and a linearly spaced vector Z of size the number of final unknowns, replaces the -\/1 by the unknowns. The final vector maps -\/1 for DOFs that are eliminated (not in the \textquotesingle{}sec\textquotesingle{} positions) and linearly lists unknown numbers for the other DOFs

2) insert, in a vector already containing values for Dirichlet DOFs, values calculated by solving a system on the other DOFs. \mbox{\Hypertarget{group__BoundaryConditions_ga1c30c96a168297b0c93e4db27d985578}\label{group__BoundaryConditions_ga1c30c96a168297b0c93e4db27d985578}}
