Vec\+Type replace\+Sections\+Vector (\begin{DoxyParamCaption}\item[{const Vec\+Type \&}]{V,  }\item[{const Vec\+Type \&}]{Z,  }\item[{const std\+::vector$<$ std\+::pair$<$ size\+\_\+t, size\+\_\+t $>$$>$ \&}]{sec }\end{DoxyParamCaption})}



Replace sections of vector V by values from vector Z into vector V; the sections are determined by \textquotesingle{}sec\textquotesingle{}. The vectors are any (identical) types of Eigen\+::\+Vector. 

Returns a vector of same length as V, whose segments starting at sec\mbox{[}i\mbox{]}.first and with length sec\mbox{[}i\mbox{]}.second are replaced using the values (successively) of Z. Can be used to\+:

1) create a map from D\+O\+Fs to unknowns\+: starting from a vector V of -\/1 of size nb of D\+O\+Fs and a linearly spaced vector Z of size the number of final unknowns, replaces the -\/1 by the unknowns. The final vector maps -\/1 for D\+O\+Fs that are eliminated (not in the \textquotesingle{}sec\textquotesingle{} positions) and linearly lists unknown numbers for the other D\+O\+Fs

2) insert, in a vector already containing values for Dirichlet D\+O\+Fs, values calculated by solving a system on the other D\+O\+Fs. \mbox{\Hypertarget{group__BoundaryConditions_gabaeb77c2484042f248639f1806fe8e72}\label{group__BoundaryConditions_gabaeb77c2484042f248639f1806fe8e72}}
