Detailed Description

template<BasisFunctionE BasisFunction>
struct HArDCore2D::evaluate_quad< BasisFunction >

Evaluate a basis at quadrature nodes. 'BasisFunction' determines what kind of value we want to evaluate (the function value of the basis, their gradients, etc.).
