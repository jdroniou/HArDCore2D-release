Directory: HArDCore2D/Schemes/HHO-full-gradient-diff

Implementation of the HHO scheme with full gradient for the diffusion problem, as per the principles described in

 The Hybrid High-Order Method for Polytopal Meshes: Design, Analysis, and Applications. 
 D. A. Di Pietro and J. Droniou. Modeling, Simulation and Applications, vol. 19. 
 Springer International Publishing, 2020, xxxi + 525p. doi: 10.1007/978-3-030-37203-3. 
 url: https://hal.archives-ouvertes.fr/hal-02151813.

If you use these codes for a scientific publication, please mention the above book.
